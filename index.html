📘 Edge 插件完整使用步骤

1

安装 Edge 扩展

  1. 打开 Microsoft Edge 浏览器（桌面版）
  2. 点击上方按钮访问 Edge 外接程序商店，或手动搜索 “ChatExport for Kimi”
  3. 在扩展详情页点击 “获取” → “添加扩展”，几秒即可安装完成
  4. 浏览器右上角工具栏出现 🍊 Kimi 小图标，扩展自动就绪
💡 安装后建议固定扩展：点击地址栏右侧拼图图标 🧩，找到“ChatExport for Kimi”并点击图钉固定。
2

单轮对话 · 即刻导出

  1. 访问 kimi.com 并登录，进入任意对话
  2. 每条 AI 回答下方会自动显示导出工具栏：
    📝 Markdown / 🌐 HTML / 📊 JSON / 📄 Word 四个快捷按钮
  3. 点击右侧 “···” 更多按钮 可展开 PDF / PNG / TXT / Jupyter Notebook (ipynb)
  4. 点击任意格式，立即下载当前这一轮问答（保留完整上下文与代码块）
✨ 鼠标悬停按钮可查看格式说明，深色模式自动适配，数学公式、表格完美保留。
3

批量导出 · 全会话归档

  1. 在 Kimi 任意对话页面，点击右上角 蓝色悬浮按钮 “📥 导出多轮对话”
  2. 弹窗内灵活勾选导出内容：全部信息 / 仅提问 / 仅回答 / 仅思考链 （支持多选）
  3. 选择导出格式（八种任意一种），可选包含话题标题+时间戳
  4. 点击 “确定导出”，一键打包下载整个会话（支持200+轮超长对话）
🚀 特别适合课程整理、论文素材归档、知识库构建，批量导出速度极快，对话结构完整。
